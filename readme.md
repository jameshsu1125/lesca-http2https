# Installation

```sh
$ npm install lesca-http2https --save
```

# Usage

```javascript
import Http2https from 'lesca-http2https';
Http2https.go();
```

# Methods

| method | options |     description     | default |
| :----- | :-----: | :-----------------: | ------: |
| go()   |         | redirect url to SSL |         |
